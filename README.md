# Умный Планировщик

Веб-приложение для планирования задач с подсказками ИИ.

## Технологии
- Vite + React + TypeScript
- shadcn-ui, Tailwind CSS

## Локальный запуск
```sh
npm i
npm run dev
```

## Сборка
```sh
npm run build
npm run preview
```

## Конфигурация окружения
Создайте файл `.env` в корне проекта и задайте переменные:
```
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_OPENROUTER_API_KEY=
```
